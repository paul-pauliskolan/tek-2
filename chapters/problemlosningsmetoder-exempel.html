Kapitel 3 - extra sida

Problemlösningsmetoder - med exempel

Kort och konkret översikt med exempel från webb och Python.

Tillbaka till kapitel 3 Till startsidan

1. Rubber Ducking

Förklara problemet högt.

  • Webb: "Min knapp funkar inte... den ska köra en funktion... men jag kanske inte har kopplat onclick?"
  • Python: "Programmet kraschar... jag skrev in text... men jag kanske inte hanterar fel i input()?"

2. Dela upp problemet

Bryt ner i mindre delar.

  • Webb: Får jag klick på knappen? Körs JavaScript? Ändras HTML?
  • Python: Läser programmet input()? Uppdateras variabler? Körs rätt villkor?

3. Arbeta baklänges

Börja från målet.

  • Webb: Mål: Visa text -> behöver funktion -> behöver event -> behöver knapp.
  • Python: Mål: Programmet ska ge rätt svar -> behöver input -> behöver beräkning -> behöver utskrift.

4. Testa med enkla exempel

  • Webb: Byt ut avancerad kod mot: console.log("funkar?");
  • Python: print("input mottagen")
  • Funkar detta? Då vet du var felet inte är.

5. Steg-för-steg (Tracing)

  • Webb: Följ vad som händer när du klickar: console.log("1"); console.log("2");
  • Python: Kolla värden: print(anvandar_input)

6. Rita upp problemet

  • Webb: Knapp -> JS-funktion -> ändra DOM.
  • Python: Input -> Funktion -> Variabel -> Resultat.

7. Pseudokod

  • Webb: Om knapp klickas: visa text.
  • Python: Om användaren skriver "ja": kör funktionen.

8. Googla smart

  • Webb: "javascript button click not working"
  • Python: "python input valueerror example"

9. Läs felmeddelanden

  • Webb: Console: Uncaught TypeError: ... -> något är undefined.
  • Python: ValueError -> fel datatyp i inmatning.

10. Ändra en sak i taget

  • Webb: Ändra bara CSS - inte HTML + JS samtidigt.
  • Python: Testa bara en funktion - inte hela programmet samtidigt.

11. Debugverktyg

  • Webb: DevTools: Inspect element, Console.
  • Python: Breakpoints i VS Code och print() för att se värden.

12. Jämför med fungerande kod

  • Webb: Din kod: onclick="myFunction" - fungerande: onclick="myFunction()"
  • Python: Din kod saknar: import random

13. Be om hjälp

  • Webb: "Min knapp gör inget när jag klickar, här är min kod."
  • Python: "Min funktion körs inte trots att jag anropar den."

14. Ta en paus

  • Webb: Efter paus: "Just det... jag glömde länka JS-filen!"
  • Python: Efter paus: "Jag glömde anropa funktionen..."

15. Bygg en minimal version

  • Webb: Bara: <button onclick="alert('hej')">Test</button>
  • Python: Bara: if alder >= 18: print("Vuxen")

16. Använd analogier

  • Webb: DOM = ett träd (du plockar grenar).
  • Python: Funktion = ett "recept" med instruktioner.

17. Gissa och testa (kontrollerat)

  • Webb: "Jag tror felet är i CSS -> testar att ta bort style."
  • Python: "Jag tror felet är i villkoret -> testar att ändra if-satsen."

18. Kontrollera antaganden

  • Webb: "Jag antar att JS-filen är kopplad... men är den det?"
  • Python: "Jag antar att variabeln har rätt värde... men har den det?"

AI som en punkt i processen

  • AI: Beskriv problemet kort, visa minimal kod och be om 2-3 hypoteser att testa en i taget.
